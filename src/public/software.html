GROMACS

GROMACS (GROningen MAchine for Chemical Simulations) es un programa de alto rendimiento, de código abierto, multiplataforma utilizado para realizar la simulación de la dinámica molecular de sistemas con cientos de millones de partículas. Gromacs fue originalmente desarrollado en la Universidad de Groningen, pero por ser de código abierto ha sido reimplementado por muchos otros desarrolladores.

FFTW es una biblioteca de subrutinas C para calcular la transformada discreta de Fourier (DFT) en una o más dimensiones, de tamaño de entrada arbitrario, y de datos tanto reales como complejos (así como de datos pares / impares, es decir, las transformaciones discretas de coseno / seno o DCT / DST).

Open MPI

El proyecto Open MPI es una implementación de interfaz de paso de mensajes de código abierto desarrollada y mantenida por un consorcio de socios académicos, de investigación y de la industria. Por lo tanto, Open MPI puede combinar la experiencia, las tecnologías y los recursos de toda la comunidad de computación de alto rendimiento para construir la mejor biblioteca de MPI disponible. Open MPI ofrece ventajas para los proveedores de sistemas y software, desarrolladores de aplicaciones e investigadores de ciencias de la computación.

HADOOP

HDFS es un sistema de archivos distribuido de Hadoop, tolerante a fallos y diseñado para hardware económico. Optimizado para grandes volúmenes de datos, relaja requisitos POSIX para alto rendimiento.

NetCDF

NetCDF (Network Common Data Form) es un conjunto de interfaces para acceso a datos orientado a matrices y una colección de bibliotecas de acceso a datos distribuidas libremente para C, Fortran, C ++, Java y otros idiomas. Las bibliotecas de netCDF admiten un formato independiente de la máquina para representar datos científicos. Juntas, las interfaces, las bibliotecas y el formato admiten la creación, el acceso y el intercambio de datos científicos.

  • Sitio oficial

  • Sitio oficial

ScaLAPACK

ScaLAPACK es una biblioteca de rutinas de álgebra lineal de alto rendimiento para máquinas de memoria distribuida en paralelo. ScaLAPACK resuelve sistemas lineales densos y en bandas, problemas de mínimos cuadrados, problemas de valores propios y problemas de valores singulares. Las ideas clave incorporadas en ScaLAPACK incluyen el uso de 1._una distribución de datos de bloques cíclicos para matrices densas y una distribución de datos de bloques para matrices en bandas, parametrizable en tiempo de ejecución; 2._algoritmos de partición en bloque para garantizar altos niveles de reutilización de datos; 3._Componentes modulares de bajo nivel bien diseñados que simplifican la tarea de paralelizar las rutinas de alto nivel haciendo que su código fuente sea el mismo que en el caso secuencial.

  • Sitio oficial
  • Versión: scalapack-2.0.2.tgz
